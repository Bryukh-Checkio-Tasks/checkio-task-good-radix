In mathematical numeral systems, the radix or base is the number of unique digits, including zero, that a positional numeral system uses to represent numbers. For example, for the decimal system the radix is ten, because it uses the ten digits from 0 through 9. In the system with radix 13, for example, a string of digits such as 398 denotes the decimal number 3 * 13^2 + 9 * 13^1 + 8 * 13^0. For the systems with the radix more than 10, we will use capital Latin symbols from A to Z, where A = 10, B = 11 ...
